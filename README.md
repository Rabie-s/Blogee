
# Bloge

A blog system was development using PHP laravel framework.




## Run the project

To run this project

```bash
  git clone https://github.com/Rabie-s/Blogee.git
  composer install
  npm install
  npm run build
  cp .env.example .env
  php artisan key:generate
  php artisan migrate
  php artisan db:seed --class=StartSeeder
  php artisan serve
```


## Run
